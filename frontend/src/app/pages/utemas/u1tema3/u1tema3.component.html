En R, la multiplicación tiene las siguientes propiedades:

Propriedades de la multiplicación de números reales. ∀ a,b,c ϵ R
1 Clausura a ∙ b ∈ R
2 Conmutativa a ∙ b = b ∙ a
3 Asociativa ( a ∙ b) ∙ c = a ∙ (b ∙ c)
4 Elemento neutro a ∙ 1 = a
5 Elemento absorbente a ∙ 0 = 0
6 Elemento inverso a ∙ 1 / a = 1
7 Distributiva a ∙ ( b ± c) = a ∙ b ± a ∙ c
